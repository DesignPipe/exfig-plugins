# exfig-config-review

Reviews `exfig.pkl` configuration for errors, warnings, and optimization opportunities.

## Trigger Phrases

- "Review my ExFig config"
- "Check exfig.pkl"
- "Optimize my config"
- "Is my ExFig config correct?"

## What It Checks

- Missing required fields (file IDs, collection names)
- Package URI version mismatch
- Empty entries
- DRY violations
- Platform coverage gaps
- Best practice adherence
