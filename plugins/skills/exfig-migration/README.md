# exfig-migration

Migration guide between major ExFig versions.

## Trigger Phrases

- "Upgrade ExFig"
- "Migrate ExFig to v2"
- "Breaking changes in ExFig"
- "Update ExFig config"

## Supported Migrations

- **v1.x → v2.x**: YAML to PKL config, Variables API, new CLI syntax
- **v2.x minor updates**: Package URI updates, new features
- **v2.x → v3.x**: (template for future)
