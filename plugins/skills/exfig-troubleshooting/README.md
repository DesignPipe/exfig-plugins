# exfig-troubleshooting

Diagnoses ExFig errors with an error catalog and guided troubleshooting steps.

## Trigger Phrases

- "ExFig is not working"
- "Export failed"
- "Figma API error"
- "PKL error"
- Any ExFig error message

## Error Categories

- Authentication (token, 401)
- Figma API (404, 429, rate limits)
- PKL configuration
- Build and environment
- Export issues
- MCP server
