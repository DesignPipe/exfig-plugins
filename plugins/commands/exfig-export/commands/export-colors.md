# Export Colors

Export color resources from Figma to platform-specific code.

## Steps

1. Check that `exfig.pkl` exists in the current directory:
```bash
ls exfig.pkl
```

2. If ExFig MCP is available, validate config first:
- Use `exfig_validate` tool

3. Run the export:
```bash
exfig colors -i exfig.pkl
```

If the user wants to use cache to skip unchanged exports:
```bash
exfig colors -i exfig.pkl --cache
```

4. Show the user which files were generated by checking git status:
```bash
git status --short
```

5. If the export fails with an error, suggest running the `exfig-troubleshooting` skill.
