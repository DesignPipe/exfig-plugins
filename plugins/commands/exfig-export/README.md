# exfig-export

Slash commands for running ExFig exports.

## Commands

| Command | Description |
|---------|------------|
| `/export-colors` | Export colors from Figma |
| `/export-icons` | Export icons from Figma |
| `/export-images` | Export images from Figma |
| `/export-all` | Batch export all resources |

Each command validates the config, runs the export, and shows generated files.
